# Hub Power & Ponto - Proposta Comercial

Landing page estática para apresentar a proposta comercial do Hub Power & Ponto.

## Como editar

- Valores: altere os campos do card `Investimento` em `index.html`.
- WhatsApp: troque `5500000000000` pelos números desejados nos dois links finais.
- Textos: todo o conteúdo comercial está centralizado no `index.html`.
- Visual: tokens de cor, espaçamento e responsividade ficam em `style.css`.

## Como publicar

### GitHub Pages

Suba estes arquivos para um repositório e habilite GitHub Pages apontando para a branch principal.

### Vercel

Importe o repositório na Vercel. Por ser uma página estática, não precisa configurar build.

## Arquivos

- `index.html`: estrutura e conteúdo da proposta.
- `style.css`: visual premium, responsividade e microinterações.
- `script.js`: animações suaves ao scroll e navegação interna.
